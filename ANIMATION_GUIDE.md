# Mystic Ring Lotto 애니메이션 기술 가이드

본 프로젝트의 메인 화면(`index.html`)에 적용된 주요 애니메이션 및 인터랙티브 기술 사양서입니다. '신비함'과 '생동감'을 극대화하기 위해 최신 웹 기술이 조화롭게 사용되었습니다.

---

## 1. 타이포그래피 애니메이션 (Interactive Typography)
**적용 대상:** `MYSTIC RING LOTTO` 메인 타이틀

*   **동적 DOM 분할 (Staggered Animation):** 자바스크립트를 사용하여 평문 텍스트를 개별 `<span>` 단위로 쪼개어 각각에 독립적인 애니메이션 딜레이(`--delay`)를 부여했습니다.
*   **플로팅 웨이브 (Floating Wave):** `@keyframes mysticFloat`를 통해 글자들이 물결치듯 위아래로 부드럽게 움직이는 효과를 주어 정적인 느낌을 제거했습니다.
*   **반응형 호버 (Interactive Hover):** 마우스를 올리면 글자가 튀어 오르며(`scale`, `translateY`), 강한 광채(`text-shadow`)를 내뿜는 물리 기반 인터랙션을 구현했습니다.

## 2. 인터랙티브 3D 틸트 (Interactive 3D Tilt)
**적용 대상:** 로또 용지(Ticket) 영역

*   **원근법 투영 (Perspective):** CSS의 `perspective: 1000px` 속성을 사용하여 입체적인 공간감을 형성했습니다.
*   **좌표 기반 회전 (Coordinate Rotation):** 마우스 커서의 위치(`clientX`, `clientY`)를 부모 요소의 상대 좌표로 계산하여, 중심점으로부터의 거리에 따라 실시간으로 `rotateX`와 `rotateY` 값을 변경합니다.
*   **레이어 분리 (Fixed vs Dynamic):** 배경의 '절대 반지'는 고정된 축으로 회전시키고, 전면의 '티켓'만 기울어지게 설계하여 시각적 깊이감(Depth)을 극대화했습니다.

## 3. 마법 입자 시스템 (Mystical Embers)
**적용 대상:** 전체 배경

*   **파티클 랜덤 생성:** JS를 통해 30개 이상의 입자를 무작위 위치에 생성하고, 비규칙적인 비행 궤적(`emberFly`)을 부여했습니다.
*   **테마 최적화 스타일링:**
    *   **다크 모드:** 갈색/금색 입자로 마법의 화염 느낌 연출.
    *   **라이트 모드:** 순백색 입자와 강한 블러(Blur) 처리를 통해 햇살에 비치는 금가루(Gold dust) 느낌으로 변환하여 벌레 같은 불쾌감을 제거했습니다.
*   **GPU 가속:** `transform` 애니메이션을 사용하여 CPU 부하를 최소화하고 부드러운 프레임을 유지합니다.

## 4. 예언 타이핑 효과 (Typewriter Prophecy)
**적용 대상:** 하단 예언 문구 섹션

*   **순차적 렌더링:** 문자열을 배열로 처리하여 특정 시간 간격으로 하나씩 출력하는 재귀적 타이핑 로직을 구현했습니다.
*   **커서 애니메이션:** CSS 의사 요소(`::after`)와 `@keyframes blink`를 조합하여 실제 타자기로 치는 듯한 생동감을 더했습니다.

## 5. 환경 시각 효과 (Environmental VFX)
**적용 대상:** 전체 배경 레이어

*   **미스틱 네뷸라 (Nebula):** 대형 방사형 그라데이션을 120초 주기로 아주 느리게 회전시켜 배경이 살아 움직이는 듯한 소용돌이 효과를 주었습니다.
*   **은하수 스타더스트 (Stardust):** 다중 배경 이미지 기법과 `stardustSlow` 애니메이션을 통해 배경이 대각선으로 끊임없이 흐르도록 구현했습니다.
*   **질감 블렌딩 (Texture Blending):** `background-blend-mode: multiply`를 테마별로 다르게 적용하여, 라이트 모드에서는 양피지 질감을, 다크 모드에서는 가죽 질감을 각각의 색상에 맞춰 최적화했습니다.

## 6. 스크롤 리빌 및 흩어짐 애니메이션 (Scroll Reveal & Scatter)
**적용 대상:** 하단 기능 카드 및 SEO 설명 섹션

*   **상태 기반 트리거 (Intersection Observer):** 브라우저의 `IntersectionObserver` API를 활용하여 요소가 화면의 특정 비율(10%) 이상 들어오거나 나갈 때 실시간으로 `.is-visible` 클래스를 토글합니다.
*   **유동적 애니메이션 (Repeatable Animation):** 일회성 애니메이션이 아닌, 사용자가 스크롤을 올리거나 내릴 때마다 반복적으로 실행되도록 설계하여 페이지 전체가 살아 움직이는 유기적인 느낌을 줍니다.
*   **복합 트랜지션 (Complex Transitions):**
    *   **나타날 때:** 아래에서 위로 올라오며(`translateY`), 흐림 효과(`blur`)가 제거되고, 살짝 비틀린 각도(`rotate`)가 정위치로 돌아오는 복합 효과 적용.
    *   **사라질 때 (흩어짐):** 요소가 다시 작아지며 아래로 가라앉고 흐릿해지는 연출을 통해 콘텐츠가 자연스럽게 배경으로 흩어지는 서사적 효과 구현.

---
**의도:** 이 기술들은 사용자가 단순히 번호를 생성하는 것을 넘어, 한 편의 판타지 서사 속에 들어온 것과 같은 몰입형 경험(Immersive Experience)을 제공합니다.
