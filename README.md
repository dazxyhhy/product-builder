# LOTTO OF THE RINGS (반지의 제왕 로또)

**"One Number to Rule Them All"**

반지의 제왕 세계관을 테마로 한 로또 번호 추천 생성기입니다. 고풍스러운 양피지와 절대 반지 디자인, 그리고 실제 로또 용지를 재현한 UI로 신비로운 번호 추첨 경험을 제공합니다.

## ✨ 주요 기능

*   **실제 로또 용지 스타일**: 한국 로또 6/45 실제 용지와 유사한 마킹 그리드 및 레이아웃 적용.
*   **반지의 제왕 테마**:
    *   신비로운 배경 음악(상상)과 시각적 분위기.
    *   배경에서 회전하는 절대 반지 애니메이션.
    *   'Cinzel' 및 '나눔명조' 폰트를 활용한 서사적 타이포그래피.
*   **인터랙티브 애니메이션**:
    *   '운명을 결정하라' 버튼 클릭 시 용지에 번호가 마킹되는 애니메이션.
    *   실제 추첨처럼 로또 공이 굴러나오는 하단 연출.
*   **운명의 예언**: 번호를 생성할 때마다 반지의 제왕 분위기의 짧은 예언 문구가 랜덤하게 나타납니다.

## 🚀 실행 방법

1. 이 저장소를 클론하거나 다운로드합니다.
2. `index.html` 파일을 웹 브라우저(Chrome, Edge, Safari 등)에서 엽니다.
3. **'운명을 결정하라'** 버튼을 눌러 당신의 운명을 확인하세요.

## 🛠 기술 스택

*   HTML5
*   CSS3 (Flexbox, Grid, Animations)
*   JavaScript (Vanilla JS)
*   Google Fonts (Cinzel, Nanum Myeongjo, Libre Barcode 39)
