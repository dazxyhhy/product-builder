# MYSTIC RING (신비한 반지)

**"운명을 확인하는 신비한 공간"**

반지의 제왕 세계관을 테마로 한 **로또 번호 생성기**를 중심으로, 다양한 운세 및 심리 테스트를 제공하는 웹 애플리케이션 모음입니다. 고풍스러운 디자인과 인터랙티브한 요소들로 사용자에게 신비로운 경험을 선사합니다.

## ✨ 주요 프로젝트

### 1. 💍 신비한 반지 로또 (Mystic Ring Lotto)
*   **파일**: `index.html`
*   **설명**: 반지의 제왕 테마가 적용된 로또 번호 추천 생성기입니다.
*   **주요 기능**:
    *   한국 로또 6/45 실제 용지 스타일의 마킹 UI.
    *   절대 반지 회전 애니메이션 및 고풍스러운 배경.
    *   번호 생성 시 나타나는 랜덤 예언 문구.
    *   다크 모드/라이트 모드 테마 지원.

### 2. 🐾 동물상 테스트 (Animal Face Test)
*   **파일**: `animal-test.html`
*   **설명**: 사용자의 얼굴형이나 특징에 맞는 동물을 찾아주는 힐링 심리 테스트 페이지입니다.

### 3. 📜 운명의 기록관 (Blog)
*   **파일**: `blog.html`
*   **설명**: 운명, 확률, 통계와 관련된 흥미로운 이야기들을 담은 블로그 페이지입니다.

### 4. 기타 스토리 페이지
*   `story-ring-power.html`: 반지의 힘에 대한 이야기.
*   `story-probability.html`: 확률의 비밀.
*   `story-lucky-habits.html`: 운을 부르는 습관.

## 🛠 기술 스택

*   **Core**: HTML5, CSS3, JavaScript (Vanilla)
*   **Styling**: Flexbox, Grid, CSS Animations
*   **Fonts**: Google Fonts (Cinzel, Nanum Myeongjo, Libre Barcode 39)
*   **Deployment**: Static Web Hosting (GitHub Pages, Cloudflare Pages, etc.)

## 🚀 실행 방법

1. 이 저장소를 클론하거나 다운로드합니다.
   ```bash
   git clone https://github.com/your-username/product-builder.git
   ```
2. `product-builder` 폴더로 이동합니다.
3. `index.html` 파일을 웹 브라우저(Chrome, Edge, Safari 등)에서 실행하면 메인 페이지(신비한 반지 로또)를 확인할 수 있습니다.
